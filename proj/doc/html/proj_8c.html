◆ gameFlow()

int gameFlow ( char *  string)

Main function to handle the game. It will control the remaining time as well as the attempts remaining. If the player loses (runs out of lives or time), it will call the function gameOver. If the player wins (gets all the letters right), it will call the function victoryScreen. As the player tries the different letters, it will draw them in the adequate position.

Parameters
string-> word to be guessed
Returns

◆ gameOver()

int gameOver ( char *  string)

If the player ran out of times or lives, they will be presented with this screen. It will present the score they had before they lost (unless it was 0), and it will show what was the right word. It will also present the options to restart the game or quit

Parameters
string-> word that was supposed to have been guessed
Returns
0 if everything went OK, 1 otherwise
